# Todo App - Spring Boot

## Project Description
This is a Todo Application built using Spring Boot, MySQL, HTML, CSS, and JavaScript.

## Features
- Add Task
- Edit Task
- Delete Task
- Mark Task as Completed

## Technologies Used
- Java (Spring Boot)
- MySQL
- HTML, CSS, JavaScript

## How to Run
1. Download the ZIP file
2. Extract it
3. Open in Eclipse
4. Run the application
5. Open browser: http://localhost:8080

## Note
Project files are uploaded as ZIP. Please download and extract to view complete source code.
